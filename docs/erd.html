ผังโครงสร้างฐานข้อมูลร้านค้าออนไลน์ wha_ecommerce

MariaDB 11.8

แผนภาพความสัมพันธ์ (Entity-Relationship Diagram)

erDiagram categories ||--o{ products : "has" products ||--o{ product_images : "has" customers ||--o{ orders : "places" orders ||--|{ order_items : "contains" products ||--o{ order_items : "is ordered in" categories { int id PK "Auto Increment" varchar name "ชื่อหมวดหมู่สินค้า" } products { int id PK "Auto Increment" varchar name "ชื่อสินค้า" text description "รายละเอียดสินค้า" double price "ราคาสินค้า" int category_id FK "หมวดหมู่สินค้า" } customers { int id PK "Auto Increment" varchar name "ชื่อ-นามสกุล" varchar address "ที่อยู่จัดส่ง" varchar phone "เบอร์โทรศัพท์" } orders { int id PK "Auto Increment" datetime date "วันที่สั่งซื้อ" int customer_id FK "รหัสลูกค้า" enum status "delivered, received, processing" double total_amount "ราคารวม" } order_items { int id PK "Auto Increment" int order_id FK "รหัสการสั่งซื้อ" int product_id FK "รหัสสินค้า" int quantity "จำนวนสินค้า" double price "ราคา ณ วันที่สั่งซื้อ" } product_images { int id PK "Auto Increment" int product_id FK "รหัสสินค้า" text image_name "ชื่อไฟล์รูปภาพ" timestamp created_at "สร้างเมื่อ" }

พจนานุกรมข้อมูล (Data Dictionary)

categories หมวดหมู่สินค้า
Field Type Key Description
id INT PK (AI) รหัสหมวดหมู่สินค้า
name VARCHAR(255) - ชื่อหมวดหมู่สินค้า เช่น สมาร์ทโฟน
products รายการสินค้า
Field Type Key Description
id INT PK (AI) รหัสสินค้า
name VARCHAR(255) - ชื่อสินค้า
description TEXT - รายละเอียดสินค้า
price DOUBLE - ราคาสินค้า
category_id INT FK เชื่อมโยงไปยัง categories(id)
customers ลูกค้า
Field Type Key Description
id INT PK (AI) รหัสลูกค้า
name VARCHAR(255) - ชื่อ-นามสกุลลูกค้า
address VARCHAR(500) - ที่อยู่จัดส่งของลูกค้า
phone VARCHAR(20) - เบอร์โทรศัพท์ติดต่อ
orders คำสั่งซื้อ
Field Type Key Description
id INT PK (AI) รหัสคำสั่งซื้อ
date DATETIME - วันที่และเวลาที่สั่งซื้อ
customer_id INT FK เชื่อมโยงไปยัง customers(id)
status ENUM - สถานะ: 'delivered', 'received', 'processing'
total_amount DOUBLE - ยอดเงินรวมในคำสั่งซื้อ
order_items รายการสินค้าในคำสั่งซื้อ
Field Type Key Description
id INT PK (AI) รหัสรายละเอียดสินค้าของคำสั่งซื้อ
order_id INT FK เชื่อมโยงไปยัง orders(id) [ON DELETE CASCADE]
product_id INT FK เชื่อมโยงไปยัง products(id)
quantity INT - จำนวนที่สั่งซื้อ
price DOUBLE - ราคาสินค้า ณ วันที่สั่งซื้อ
product_images รูปภาพสินค้า
Field Type Key Description
id INT PK (AI) รหัสรูปภาพสินค้า
product_id INT FK เชื่อมโยงไปยัง products(id) [ON DELETE CASCADE]
image_name TEXT - ชื่อไฟล์รูปภาพสินค้า
created_at TIMESTAMP - วันที่อัปโหลดรูปภาพ
